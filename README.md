# ClassHelper 项目说明

## 概述
ClassHelper 是一个用于辅助课堂教学的 Python 工具。

## 功能特点
- **随机选人**: 从预设的学生名单中随机选择一名学生。
- **计时器**: 提供一个简单的计时器功能，支持开始、暂停和设置时间。

## 使用指南
1. 确保你已经安装了 Python 3.6 或更高版本。
2. 克隆项目仓库到本地:
   ```bash
   git clone https://github.com/lzm222/classhelper.git
3. 进入项目目录并运行程序:
   ```bash
   cd classhelper
   python main.py
